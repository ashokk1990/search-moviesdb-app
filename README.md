## Available Scripts

In the project directory, you can run:

### `yarn install`

### `yarn start`

### `yarn build`


You can now view search-moviesdb-app in the browser.

`  Local:            http://localhost:3000/
   On Your Network:  http://yourIP:3000/ `

Note: that the development build is not optimized.
